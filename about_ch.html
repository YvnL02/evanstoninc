公司简介

Evanston Pharma Inc 成立于2003年，并于 2012 年更名为 Evanstoninc LLC

最初专注于 API 销售，最近将重心转移到进入美国市场的咨询和制剂产品。 公司已帮助众多中国公司获得 FDA 批准并为其 API 产品建立销售。 此外，两家中国公司的制剂产品通过Evanstoninc为代理成功进入美国市场，销售 稳定。 其中一家是浙江的上市公司，另一家位于江苏， 还有一家浙江上市公司的 产品正在等待FDA现场检查并完成销售合同。
